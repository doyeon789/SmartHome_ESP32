# SmartHome_ESP32

### 전송 방식 esp32

 LED0 / LED1 / LED2 / LED3 / LED4 / LED5 / LED6 / LED7 / LED8 / LED9 / LEDa / LEDb / RGB / Motor0 / Motor1 \n
"0000000000000000"

RGB -> 
0 : 꺼짐 
1 : 초록 
2 : 빨강 

Motor0 -> 
0 : 0도 (닫힘)
1 : 90도 (열림)

Motor1 ->
0: 꺼짐
1: 미풍 (60)
2: 약풍 (80)
3: 강풍 (100)
